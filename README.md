# Building Project

A comprehensive project management application for construction and building projects.

## Features

- Project management with phases (Design, Build, Certification)
- Stage and module tracking
- File management and document preview
- Cost tracking and payments
- Template management
- User permissions and access control
- Approval workflows

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Drizzle ORM
- Cloudflare Workers
- Cloudflare D1 (Database)
- Cloudflare R2 (File Storage)
- Clerk (Authentication)
